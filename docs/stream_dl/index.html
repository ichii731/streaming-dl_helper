Streaming-DL Helper

難しい操作不要でABEMAやニコ生等のストリーミング保存が行える補助ツールです.

ダウンロード (Installer v1.0)

exe形式/Win11対応

使い方

minyami拡張を導入

まずChrome拡張機能であるminyamiをインストールします.

インストールページ

コマンドをコピー

保存したいストリーミング再生ページにアクセスして,拡張機能アイコンをクリック.「复制」でコピーします.

ソフトで保存処理開始

本ソフトウェアを起動して,コマンドを貼り付けて「処理開始」を押すと保存が開始されます.下部でログを確認可能.
